# 双语翻译助手 - Chrome 扩展

一个在网页上实现双语对照翻译的浏览器扩展。

## 功能特性

- **整页翻译**: 一键翻译当前页面的所有内容
- **选中翻译**: 仅翻译用户选中的文本内容
- **双语对照**: 支持原文和译文对照显示
- **多语言支持**: 支持中文、英文、日文、韩文、法文、德文等
- **自定义设置**: 可配置目标语言、翻译服务等选项

## 安装方法

### 开发模式安装

1. 打开 Chrome 浏览器
2. 在地址栏输入: `chrome://extensions/`
3. 开启右上角的 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"** 按钮
5. 选择本项目的根目录 (`2026-03-27-browser-translator/`)
6. 扩展安装完成，右上角会出现翻译图标

## 使用方法

1. **访问任何网页** (如 https://example.com)
2. **点击右上角翻译图标**，弹出操作面板
3. **选择翻译模式**:
   - "翻译整页": 翻译当前页面的所有内容
   - "翻译选中内容": 仅翻译您选中的文本
4. **查看结果**: 翻译结果会显示在控制台 (按 F12 打开开发者工具)

## 开发说明

### 技术栈
- Manifest V3 (Chrome 扩展最新规范)
- Vanilla JavaScript (无框架依赖)
- HTML5 + CSS3
- Chrome Extension APIs

### 核心文件说明

#### manifest.json
扩展的配置文件，定义:
- 扩展名称、版本、描述
- 所需权限 (storage, activeTab, scripting)
- 后台服务脚本 (Service Worker)
- 内容脚本注入规则
- 弹出窗口和设置页面配置

#### content.js
注入到每个网页中的脚本，负责:
- 监听来自 popup 的翻译请求
- 处理网页内容提取和翻译结果展示
- 在控制台输出调试信息
- 显示用户通知

#### popup.js
弹出窗口的逻辑处理:
- 用户交互处理 (按钮点击)
- 设置保存和加载
- 与当前标签页通信
- 状态显示和反馈
