## 安装说明
1. 解压此压缩包
2. 打开 Chrome: chrome://extensions/
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择解压后的目录

## 注意事项
- 需要 Chrome 88+ 版本
- 部分 Chrome 特殊页面不支持扩展
- 首次使用需在设置页面配置
